Auftragsallokation

Hier können sie eine Tour erstellen. Wählen Sie zunächst ein Fahrzeug aus, das die Tour fahren soll. Danach erhalten Sie einen Überblick, über alle Aufträge, die noch keiner Tour zugeordnet wurden und somit ausgewählt werden können. Wenn Sie fälschlicherweise einen Auftrag ausgewählt haben, können Sie diesen durch einen Klick auf das Minus-Symbol links in der Tabelle wieder entfernen.
Bisher gibt es eine Beschränkung der Aufträge auf maximal fünf pro Tour.
Wenn sie auf "Speichern" drücken, wird automatisch geprüft, ob das zulässige Ladungsgewicht des Fahrzeugs dem Gesamtgewicht der Ladungen der Aufträge entspricht. Außerdem wird die Gesamtlänge, -breite und -höhe der einzelnen Ladungen mit den Fahrzeugdaten verglichen.

Aufträge

Hier erhalten Sie einen schnellen Überblick über alle Aufträge.
Außerdem können sie entweder über das Menü oder über den Link in der Auftragstabelle einen neuen Auftrag anlegen. Hierzu werden verschiedene Daten abgefragt. Der Kunde wird anhand der eingegebenen Kunden-Id gesucht. Die Eingabefelder sind teilweise auf nummerische Eingaben beschränkt, um fehlerhafte Eingaben zu verhindern. Außerdem wurde bei den nummerischen Eingabefeldern ein Maximum angegeben, um unsinnige Eingaben nicht zuzulassen.

Tour Berechnen

Hier finden Sie ein Dropdown-Menü, in dem Sie eine Tour-Id auswählen können. Zu der entsprechenden Tour wird im Anschluss, wenn sie auf den Button drücken, die optimape Reihenfolge der Aufträge und die optimale Tour ausgegeben.
